      About this build

      Densa Deck v-

      Analyst model (optional, Pro)

      Local LLM for executive summary + cut suggestions. Runs fully offline after download. Choose 0.5B (~400 MB, faster) or 3B (~1.8 GB, better prose).

      Moxfield blocks direct imports

      Moxfield's Cloudflare gate rejects automated imports. The fastest way to load this deck is the built-in Export → Text feature — about 10 seconds:

      1. Open the deck on moxfield.com —
      2. On the deck page, click More → Export → Text
      3. Copy the list (Ctrl+A then Ctrl+C) and paste it into the Paste a decklist box below.

      Archidekt URL imports work directly — paste those instead if you have a copy of the deck on Archidekt.
